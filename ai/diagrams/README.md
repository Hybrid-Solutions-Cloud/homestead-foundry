# diagrams/

Exports and an index for the Lucid diagrams. The living diagrams are in Lucid under **My Documents / Microsoft AI Foundry**. Authored by `foundry-diagrammer` (Opus) and layout-checked by `foundry-diagram-qa` (Sonnet).

This folder holds the exported PNGs and an index table (diagram name, Lucid share link, PNG file) once diagrams pass QA.

Note: this initiative uses Lucid at the owner's direction, which is a documented deviation from the default draw.io diagram standard.
